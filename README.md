# Educlash : Bellum Lycei Privati et Publici

## Description

Ce projet a pour objectif d'étudier les différences de performance entre les lycées publics et les lycées privés sous contrat en France. Dans le cadre du module gestion de projet de la licence MIASHS, nous utiliserons des outils de data science tels que l'Analyse en Composantes Principales (ACP), le clustering, la classification, la prédiction, etc. pour analyser les données disponibles.

## Problématique

**Quelle structure (lycée publique ou lycée privée sous contrat) présente les meilleures performances sur le territoire français métropolitain ?**

## Données et Méthodologie

- **Données** : Les données utilisées dans ce projet proviennent du site officiel du ministère de l'Éducation nationale [Données des lycées](https://data.education.gouv.fr/explore/embed/dataset/fr-en-indicateurs-de-resultat-des-lycees-denseignement-general-et-technologique/table/?sort=-presents_gnle).
  
- **Méthodologie** :
    - Clustering
    - ACP (Analyse en Composantes Principales)
    - Statistiques descriptives
    - Statistiques inférentielles
    - Classification
    - Prédiction


## Résultats attendus

Ce projet est un exercice éducatif visant à approfondir mes compétences en data science, et non à produire des conclusions sociologiques profondes. L'objectif est de comprendre les techniques de data science tout en appliquant des méthodes sur un ensemble de données réelles.
